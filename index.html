Panduan Penggunaan

Selamat datang di aplikasi UAS Virtual Reality Butterfly_AR. Berikut adalah panduan untuk menggunakan aplikasi ini:

  • Klik tombol "Start" pada halaman utama untuk memulai aplikasi AR.
  • Arahkan kamera Anda ke AR Marker (gambar yang telah dicetak) untuk melihat model 3D kupu-kupu.
  • Setiap AR Marker akan menampilkan model kupu-kupu yang berbeda.
  • Pastikan pencahayaan cukup dan kamera fokus pada AR Marker untuk hasil terbaik.

Tentang Website Ini

Website ini dibuat sebagai bagian dari Ujian Akhir Semester (UAS) untuk mata kuliah Pemrograman Teknologi Informasi tahun 2021. Aplikasi ini menggunakan teknologi Augmented Reality (AR) untuk menampilkan model 3D kupu-kupu.

Pembuat: Dewi Ria Latifah, PTI 2021.
